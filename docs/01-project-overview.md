## Project Overview

Due or Die is a web-based task management application designed to help users focus on what truly matters by encouraging commitment and accountability.

The platform allows users to organize tasks using both a simple list view and a Kanban board, offering flexibility for different productivity styles while keeping the experience minimal, fast, and goal-oriented.
